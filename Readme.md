# 🧠 AI Mental Health Assistant – Suicide Risk Detection System

## 📌 Overview

The AI Mental Health Assistant is a soft-computing-based web application designed for the early detection of suicide risk from user chat behavior. The system analyzes text messages using Natural Language Processing (NLP) and Machine Learning techniques to classify messages as **Suicide Risk** or **Non-Risk** and provides confidence scores along with visual analytics.

The project is developed as part of a Soft Computing research study titled:

**"Early Suicide Risk Detection from Chat Behaviour"**

---

## 🎯 Features

* Real-time suicide risk prediction
* Text and voice input support
* TF-IDF feature extraction
* Support Vector Machine (SVM) classification
* Confidence score generation
* Interactive analytics dashboard
* Risk distribution visualization
* Voice feedback using speech synthesis
* FastAPI backend
* React frontend

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/your-username/suicide-risk-detection.git
cd suicide-risk-detection
```

### Backend Setup

```bash
cd backend

python -m venv .venv

.venv\Scripts\activate

pip install -r requirements.txt

uvicorn app:app --reload
```

Backend runs on:

```bash
http://127.0.0.1:8000
```

### Frontend Setup

```bash
cd frontend

npm install

npm start
```

Frontend runs on:

```bash
http://localhost:3000
```
